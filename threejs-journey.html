01

Empty Canvas

Every 3D world needs three things:

Scene = the stage. Camera = your eyes. Renderer = the painter that draws what the camera sees onto the screen.
In Blender you already have these — a scene, a camera, and the viewport renderer. Three.js makes you set them up with code.
Step 1 — Paste inside your script tag 
import * as THREE from 'three';

const canvas = document.getElementById('canvas');
const w = window.innerWidth;
const h = window.innerHeight;

// Scene — the stage
const scene = new THREE.Scene();
scene.background = new THREE.Color(0x1a1a2e);

// Camera — your eyes
const camera = new THREE.PerspectiveCamera(50, w / h, 0.1, 100);
camera.position.set(0, 2, 5);
camera.lookAt(0, 0, 0);

// Renderer — the painter
const renderer = new THREE.WebGLRenderer({ canvas, antialias: true });
renderer.setSize(w, h);

renderer.render(scene, camera);

05

Light

MeshBasicMaterial ignores light — that's why it looks flat. MeshStandardMaterial reacts to light like real surfaces.

AmbientLight = overcast sky (soft, everywhere). DirectionalLight = the sun (parallel rays, creates highlights and shadows).
MeshStandardMaterial = Blender's Principled BSDF. Same PBR model, same realistic result.
Step 5 — Swap material + add lights 
// Upgrade material — reacts to light
const material = new THREE.MeshStandardMaterial({
    color: 0x8b5cf6,
    metalness: 0.3,
    roughness: 0.5
});

// Ambient light (soft fill)
scene.add(new THREE.AmbientLight(0xffffff, 0.4));

// Directional light (the sun)
const sun = new THREE.DirectionalLight(0xffffff, 0.8);
sun.position.set(5, 5, 5);
scene.add(sun);

06

Materials

MeshPhysicalMaterial gives you everything Standard has, plus Transmission (see-through, like glass) and Clearcoat (glossy top layer, like car paint).

Core properties: Metalness (0 = plastic, 1 = chrome), Roughness (0 = mirror, 1 = chalk).

Blender: Principled BSDF > Metallic, Roughness, Transmission, Coat Weight — same controls.
Step 6 — Upgrade the material line 
// Physical Material — adds glass + clearcoat
const material = new THREE.MeshPhysicalMaterial({
    color:        0x8b5cf6,
    metalness:    0.3,
    roughness:    0.5,
    transmission: 0.0,  // 0 = solid, 1 = see-through
    clearcoat:    0.0,  // 0 = none, 1 = glossy top layer
    thickness:    0.5,
    ior:          1.5   // how light bends (1.5 = real glass)
});

07

Orbit

OrbitControls lets the viewer orbit, zoom, and pan — just like Blender's viewport.

Camera on a boom arm. Left-drag = orbit, scroll = zoom, right-drag = pan.

The scene redraws only when you drag — no animation loop yet. That comes next.

08

Animate

Still images are boring. An animation loop redraws 60 times per second.

A flipbook — each page almost identical, flip fast = smooth motion. That's requestAnimationFrame.

Animation = changing any property over time. Rotation, position, scale — anything.

Step 8 — Replace the change listener with this 
// Animation loop — runs ~60 times per second
controls.enableDamping = true;
var t = 0;

function animate() {
    requestAnimationFrame(animate);
    t += 0.01;

    // Spin — rotate on Y axis
    mesh.rotation.y += 0.01;    // Bob — float up and down
    mesh.position.y = Math.sin(t) * 0.5;    // Pulse — breathe in and out
    var s = 1 + Math.sin(t) * 0.2;
    mesh.scale.set(s, s, s);    // Wobble — tilt on two axes
    mesh.rotation.x = Math.sin(t * 1.5) * 0.3;
    mesh.rotation.z = Math.cos(t * 2) * 0.2;

    controls.update();
    renderer.render(scene, camera);
}
animate();

09

Populate

One shape is lonely. Let's fill the scene with multiple objects on a ground plane.

Like duplicating objects in Blender — each one independent, with its own position and material.

Each object also stores its own animation type and speed using userData — Blender calls this "Custom Properties".

Step 9 — Replace the single mesh with this 
// Ground plane

    new THREE.PlaneGeometry(8, 8),
    new THREE.MeshStandardMaterial({ color: 0x2a2a3e, roughness: 0.9 })
);
ground.rotation.x = -Math.PI / 2;
scene.add(ground);

// Object collection
const objects = [];

function addObject(type, col, x, y, z) {
    const m = new THREE.Mesh(
        makeGeometry(type),
        new THREE.MeshPhysicalMaterial({
            color: col, metalness: 0.3, roughness: 0.5,
            thickness: 0.5, ior: 1.5
        })
    );
    m.position.set(x, y, z);
    m.userData.animType = 'spin';
    m.userData.animSpeed = 0.01;
    m.userData.baseY = y;
    scene.add(m);
    objects.push(m);
}

// Updated animation — each object animates
var t = 0;
function animate() {
    requestAnimationFrame(animate);
    t += 0.016;
    objects.forEach(function(o) {
        var spd = o.userData.animSpeed;
        switch (o.userData.animType) {
            case 'spin':   o.rotation.y += spd; break;
            case 'bob':    o.position.y = o.userData.baseY + Math.sin(t * spd * 100) * 0.5; break;
            case 'pulse':  var s = 1 + Math.sin(t * spd * 100) * 0.2; o.scale.set(s,s,s); break;
            case 'wobble': o.rotation.x = Math.sin(t*spd*150)*0.3; o.rotation.z = Math.cos(t*spd*200)*0.2; break;
            case 'none':   break;
        }
